Vamos aprender a colocar uma raiz/root no CSS

Que representa uma configuração global, ou, uma variavel que guarda uma informação para que eu possa utilizar com facilidade

Vamos ver o exemplo acima

O primeiro fato que temos que observar, essa é uma página exclusiva de CSS, por isso na 1º linha devemos colocar o tradicional '@charset "UTF-8";'

O segundo fato a observar é o @import url na linha 3, que é uma fonte importada/embed do googlefonts

E então observamos o @font-face que é utilizado para colocarmos fontes baixadas, que no exemplo foi a fonte idroid. Detalhe pro "url(../fontes/idroid.otf')"", que os (..) utilizados antes de '/fonte' é muito importante nesse caso, pois os conteúdos desse site estavam em diferentes pastas, dessa forma os pontos servem para ele ir buscar o conteúdo em outra pasta

E então chegamos ao root, que é uma pseudo-classe, podemos ver pelo ":" antes do nome, para funcionar devemos colocar dois tracinhos "--" antes da função, como "--cor0: #ebe5c5;". Foram utilizados exemplos de cores e fontes, e para serem utilizados, devemos usar a funçao var, por exemplo: "color: var(--cor1);" ou "font-family: var(--fonte-padrao);"

É uma forma muito útil de economizar tempo pois se utilizado em várias tags, devemos apenas alterar no root, e não em todas tags que foi colocado
