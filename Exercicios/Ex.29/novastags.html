Aprendendo novas configurações

  1. Transition-duration

    2. Utilizado para duração de transições, por exemplo, '1s;' '0.5s' ou até '.5s', normalmente usado ':hover', na hora que algo aparece

  2. Text-indent

    3. Utilizado como ínicio de paragrafo tradicional, pois da espaço no inicio do texto

  3. Line-height

    4. Aumenta ou diminui o espaçamento do texto

  4. List-style-position

    5. Serve para alterar a posição das listas, 'inside' ou 'outside'

  5. Columns

    6. Serve para dividir uma lacuna em colunas, normalmente utilizado em listas

  6. List-style-type

    7. Serve para alterar o type das listas, podemos colocar qualquer emoji que quisermos, por exemplo: 'list-style-type: '\2714\00A0\00A0';' Que nesse caso o \2714 é o código do emoji, e o \00A0\00A0 são para dar espaço
